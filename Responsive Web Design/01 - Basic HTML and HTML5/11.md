# Lesson 11: Link to Internal Sections of a Page with Anchor Elements

La etiqueta __a__ tambien puede ser usadapara enlazar elementos internos, permitieno saltar a diferentes secciones de nuestra pagina.


Para crear un link interno, al href debemos indicarle el id del elemento al que queremos ir.

~~~
    <a href="#contacts-header">Contacts</a>
    ...
    <h2 id="contacts-header">Contacts</h2>
~~~
